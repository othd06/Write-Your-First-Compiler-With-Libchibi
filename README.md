A book to teach you how to write your first compiler with the [Libchibi](https://github.com/othd06/libchibi) library.

In this book we will go through the process of designing and implementing a custom language, ChibiLisp, learning the skills you need to do so as we go along.

Please Note: the book is currently work in progress and current chapters/references are all subject to change. I am pretty sure that chapters 0 through 8 are now correct (and can confirm they do produce a working compiler up to the feature set implemented) but I cannot 100% guarantee that there won't still be changes and later chapters are still very much subject to change.
